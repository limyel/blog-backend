# blog

一个简单的博客后端，使用 Java 开发，通过 docker 部署。


# 技术栈

| 名称            | 说点什么       |
|---------------|------------|
| Spring        | IoC 和 aop  |
| Spring MVC    | 请求和响应      |
| Spring Boot   | 约定优于配置     |
| MyBatis       | （伪）orm     |
| MyBatisPlus   | MyBatis 辅助 |
| MySQL         | 关系型数据库     |
| Redis         | KV 数据库     |
| ElasticSearch | 全文搜索       |
